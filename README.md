# 使用说明
将DirectoryToMindMap.exe放到主文件夹下即可生成当前文件夹下资料的思维导图，输出的思维导图文件为.mm格式，常用软件均支持打开，此处推荐使用开源项目[https://github.com/wanglin2/mind-map](https://github.com/wanglin2/mind-map)
